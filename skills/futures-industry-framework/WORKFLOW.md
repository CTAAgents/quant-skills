## 详细流程说明

### 阶段1: 数据收集与准备

#### 1.1 产业链数据
- 库存数据：期货交易所库存、社会库存
- 价格数据：期货价格、现货价格
- 开工率：上下游企业开工率
- 利润数据：上中下游利润分布

#### 1.2 基本面数据
- 供需数据：供应量、需求量
- 库存变化：环比、同比变化
- 基差数据：期货升贴水

#### 1.3 技术面数据
- K线数据：日线、周线、月线
- 成交量数据：持仓量、成交量
- 持仓结构：多空持仓变化

### 阶段2: 产业链定性分析

#### 2.1 库存周期判定

**判定规则：**
```
主动去库: 库存↓ + 价格↓ + 开工↓
被动去库: 库存↓ + 价格↑ + 开工↑
主动补库: 库存↑ + 价格↑ + 开工↑
被动补库: 库存↑ + 价格↓ + 开工↓
```

#### 2.2 利润结构分析

**判定规则：**
```
上游盈利: 上游利润率 > 中游 > 下游
中游盈利: 中游利润率 > 上游 > 下游
下游盈利: 下游利润率 > 中游 > 上游
全链亏损: 各环节利润率为负
利润均衡: 各环节利润率相近
```

#### 2.3 综合评分

**评分维度：**
- 库存维度 (25%)
- 利润维度 (30%)
- 行为维度 (20%)
- 供需维度 (15%)
- 基差维度 (10%)

### 阶段3: 基本面量化分析

#### 3.1 计算公式

```python
fundamental_score = (
    0.30 × (supply_demand_gap × 50 + 50) +
    0.25 × (50 - inventory_change × 50) +
    0.25 × (profit_margin × 50 + 50) +
    0.20 × (basis_structure × 50 + 50)
)
```

### 阶段4: 技术面择时分析

#### 4.1 评分指标

- **趋势评分**: 均线系统、趋势线
- **动量评分**: RSI、MACD
- **波动率评分**: ATR、布林带
- **成交量评分**: 量价配合
- **形态评分**: K线形态、图形识别

### 阶段5: 共振信号检测

#### 5.1 判定规则

```python
# 维度信号判定
if score > 55: signal = "看多"
elif score < 45: signal = "看空"
else: signal = "中性"

# 共振等级
if 3个维度同向: "极强共振"
elif 2个维度同向: "强共振"
elif 1个维度同向: "中等共振"
else: "弱共振"
```

#### 5.2 操作建议

| 共振等级 | 信号方向 | 建议 |
|---------|---------|------|
| 极强共振 | 看多 | 强烈建议买入 |
| 强共振 | 看多 | 建议买入 |
| 中等共振 | 看多 | 轻仓尝试买入 |
| 弱共振 | - | 观望 |
| 无共振 | - | 观望 |

### 阶段6: 权重优化

#### 6.1 优化流程

1. 收集历史信号和收益数据
2. 定义权重搜索空间
3. 执行网格搜索
4. 回测验证
5. 选择最优权重

#### 6.2 默认权重

```python
default_weights = {
    'industry_chain': 0.40,  # 产业链权重
    'fundamental': 0.30,    # 基本面权重
    'technical': 0.30       # 技术面权重
}
```

### 阶段7: 交易计划

#### 7.1 仓位管理

| 共振强度 | 置信度 > 0.8 | 置信度 < 0.8 |
|---------|-------------|-------------|
| 极强共振 | 30% | 20% |
| 强共振 | 30% | 20% |
| 中等共振 | 15% | 10% |
| 弱共振 | 0% | 0% |

#### 7.2 止损止盈

**看多场景：**
- 止损: -5%
- 止盈1: +8%
- 止盈2: +15%
- 止盈3: +25%

**看空场景：**
- 止损: +5%
- 止盈1: -8%
- 止盈2: -15%
- 止盈3: -25%

### 阶段8: 风险控制

#### 8.1 风控规则

1. **仓位限制**: 单品种最大仓位 30%
2. **止损规则**: 触发止损立即平仓
3. **回撤控制**: 最大回撤 15%
4. **信号过滤**: 置信度低于 0.6 降低仓位

#### 8.2 预警机制

- 库存周期异常预警
- 基差结构异常预警
- 持仓结构异常预警
- 波动率异常预警

## 常见问题

### Q1: 如何选择权重优化周期？
**A**: 建议使用1-2年的历史数据，每周重新优化一次权重。

### Q2: 三个维度信号冲突时怎么办？
**A**: 
- 2:1 遵循多数
- 1:1:1 以产业链为主
- 添加置信度权重

### Q3: 如何处理夜盘数据？
**A**: 
- 使用连续合约
- 区分日夜盘收益率
- 注意交易时间对齐

### Q4: 什么时候需要调整阈值？
**A**:
- 市场结构发生重大变化
- 连续出现假信号
- 回测结果与实盘差距大
