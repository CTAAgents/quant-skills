# CTAAgents Skills Repository

This repository contains skill definitions for CTAAgents.

## Skills Available

### Financial Analysis Skills

1. **crude-oil-daily-news** - 原油每日资讯推送任务
   - Provides daily crude oil market news and technical analysis
   - Includes comprehensive technical indicators: MA, RSI, MACD, ADX, CCI, ATR, OVX, etc.

2. **precious-metals-daily-news** - 贵金属每日资讯推送任务
   - Provides daily precious metals market news and technical analysis
   - Covers gold, silver, platinum, and palladium
   - Includes ETF holdings tracking and central bank gold purchases analysis

3. **noble-metal-oil-daily-news** - 贵金属原油综合每日资讯推送
   - Combined precious metals and crude oil daily news
   - Integrated RSI/CCI/OVX analysis framework

4. **futures-industry-framework** - 期货产业链分析框架
   - Comprehensive futures industry chain analysis
   - Includes industry chain standardization, integrated analysis, resonance detection

5. **agentic-factor-investing** - 因子投资框架
   - Agentic AI factor investing framework
   - Automated factor discovery and systematic investment

6. **factorengine** - 因子引擎
   - Program-level knowledge injection factor mining framework

### Utility Skills

7. **grill-me** - 深度压力测试技能
   - Systematic decision tree analysis and risk assessment
   - Helps validate plans and designs

8. **skillevolver** - 技能演化框架
   - Meta-skill framework for online skill learning

9. **embodiskill** - 具身技能框架
   - Skill-aware reflection and evolution for embodied skills

10. **skill-adaptor** - 技能适配器框架
    - Trajectory-based LLM agent adaptive skill framework with explicit failure attribution
